# CreDA Web Client App

- Deployed url: https://creda.app/

## Development

### Initial setup

- `npm i -D`

### Run for development

- `npm run start`

### Build for production

- `npm run build-all`
- A creda.zip file is created with the static prod content to deploy.

### Test production build locally before deployment

- `npm install -g serve`
- `serve -s build`
